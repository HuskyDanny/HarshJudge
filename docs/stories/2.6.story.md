# Story 2.6: Create Status Skill

## Status

Done

## Story

**As a** Claude Code user,
**I want** a status skill that reports scenario and run status,
**so that** I can see test health at a glance.

## Acceptance Criteria

1. `skills/harshjudge/tasks/check-status.md` created as executable task workflow
2. Task triggers defined in `skill.yaml` for: "test status", "scenario status", `/harshjudge:status`
3. `skills/harshjudge/templates/status-output-tmpl.md` created with formatting templates
4. Task retrieves data using `getStatus` MCP tool
5. Task detects in-progress runs from `skill-state.yaml` files
6. Formatted output includes scenario table, summary stats, recent failures
